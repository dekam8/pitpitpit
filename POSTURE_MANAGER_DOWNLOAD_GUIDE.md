# 자세 매니저 다운로드 사용 안내

## 바로 사용하기

1. `posture-manager-web.zip` 파일을 다운로드합니다.
2. 압축을 풉니다.
3. `dist/index.html` 파일을 브라우저로 엽니다.

서버 없이도 기본 기능을 사용할 수 있습니다.

## 포함된 기능

- 캐릭터 기반 자세/휴식 매니저
- 캐릭터만 보이는 미니 모드
- 10초, 30초, 1분, 50분 알림 간격
- 완료, 5분 뒤, 스킵, 일어났어요 액션
- 오늘 기록 저장
- 브라우저 localStorage 저장

## 중요한 제한

이 zip은 웹앱 배포본입니다. 브라우저 안에서 실행되므로, 다른 앱 위에 항상 떠 있는 OS 레벨 창은 아닙니다.

Codex, VS Code, Chrome 같은 다른 창 위에 캐릭터만 계속 띄우려면 Electron 또는 Tauri 데스크톱 앱으로 패키징해야 합니다.
